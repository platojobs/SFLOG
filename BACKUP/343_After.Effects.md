# [After Effects](https://github.com/platojobs/SFLOG/issues/343)

[Bodymovin](https://github.com/airbnb/lottie-web)是由 Hernan Torrisi 开发的 After Effects 插件，它可以将 After Effects 文件导出为 JSON 格式，并包含一个 JavaScript Web 播放器。我们基于他出色的工作成果，将其应用扩展到了 Android、iOS、React Native 和 Windows 平台。

[LottieFiles for After Effects](https://lottiefiles.com/plugins/after-effects)是由 LottieFiles 开发的一款插件，它不仅支持原生渲染动画，还提供其他功能，例如移动设备测试和渲染图表，让您在创建 Lottie 动画时可以查看 CPU 使用情况。此外，您还可以直接通过插件访问 LottieFiles 提供的包含数千个免费动画及其 AEP 文件的动画库。
