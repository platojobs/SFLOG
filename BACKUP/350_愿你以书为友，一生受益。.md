# [愿你以书为友，一生受益。](https://github.com/platojobs/SFLOG/issues/350)

为什么越厉害的人，越爱读书？答案太扎心了
 你有没有过这种感觉：
明明很努力，却总觉得脑子空空；
遇到一点事，就容易焦虑、暴躁、想不开；
跟人聊天，常常插不上话，一开口就词穷。
其实，你缺的不是运气，
而是读过的书、沉淀的认知、藏在骨子里的底气。
很多人以为，读书是学生时代的事，
长大后才明白：
读书，是普通人成本最低的自我升值。
书读得多了，你会发现：
- 情绪不再轻易被别人牵动
- 遇事不再只会抱怨，而是懂得思考
- 眼界宽了，格局大了，路自然就顺了
你读过的每一页书，都不会白费。
它会悄悄融进你的气质、谈吐、三观里，
在你迷茫时指路，在你低谷时托底。
别再说“没时间读书”，
真正拉开人与人差距的，
就是别人刷短视频的时间，你在悄悄看书。
从今天起，
少刷一点手机，多读一页书。
你读过的书，终会变成你的铠甲和光芒。
愿你以书为友，一生受益。
